# Term Descriptions
This file contains all the descriptions for the terms used in the AHP .yaml file

# Parent criteria
    plant: criteria relating to the plant characteristics (morphological, lifecycle, horticultural etc)
    medicinal: criteria relating to the medicinal properties of the plant quantified by various metric (e.g., clinical trials, medicinal activity).
    see: stands for social, environmental, and economic.
